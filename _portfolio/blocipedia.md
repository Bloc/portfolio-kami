## Explanation

How will you truly ever know if there is a legitimate competitor to Wikipedia unless you build one and publish it for all to see?
My goal for this project was to create a site for users to generate Wikis with markdown capabilities.  For anyone interested, a premium paid plan
is an option so their Wikis will remain private.
